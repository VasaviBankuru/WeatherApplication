## Weather App
 A simple Weather App made using Android Jetpack Compose which fetches weaher data using https://api.weatherapi.com Api

 ## Tech Stack
 * Kotlin
 * Compose
 * MVVM
 * Clean Architecture
 * Image loading - Coil
 * Network calls - Retrofit
 * Design - Material
 * JSON Parsing - Gson
